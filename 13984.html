測試環境

前面的環境建置好後，就可以直接打開習慣的程式碼編輯器，直接開始撰寫Python的程式了！ 以下用Visual Studio Code來展示。

打開Visual Studio Code

有一種方式是直接在電腦上面打開Vscode，這樣的開啟方法會無法流暢的指定製作的專案資料夾。 因此，提供另外一個方法，直接使用cmd（上一張提到的命令提示字元）開啟Vscode。 首先，如果你的專案資料夾想要放在磁碟槽的一個地方可以直接打上
cd （想打開的資料夾）
但是，如果你想要把cmd切換成任意磁碟槽的任何一個地方，就要把上述的指令改為
cd /d （任意磁碟槽）:/（想打開的資料夾）

打開虛擬環境

假設你想要在cmd內直接建立資料夾可以輸入: mkdir （你想要的名稱） 再利用cd進入資料夾內喔！

最後，用Vscode打開資料夾的話就可以直接輸入
code .

切換成中文的地方在這！
轉載自Ray https://israynotarray.com/vscode/20210704/1752541725/的文章

如果想要建立新檔的話只要按ctrl+N，或者在檔案>建立文字檔即可。

接著，按下ctrl+K後，在按下M就會出現這樣的畫面

接下來，在格子內打上Py就可以找到Python了，按下Enter後，檔名就會被變成Untitled-1.py。 然後按下crtl+S存檔後， 就來輸入Python的語法吧！ 只要在檔案上輸入
print("Hello World")
按下右上角的三角形就可以執行Python了喔！

有個小提醒，可以看到我們先前一章安裝的虛擬環境居然自動被Vscode讀到而且自動開啟了。 沒錯，Vscode會自動讀取有沒有虛擬環境後直接幫你開啟喔！
